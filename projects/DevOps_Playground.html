My DevOps Playground
The Goal
This project is to act as a jumping off point for me to apply new technologies in an sandbox closer to what I'd expect in a live environment.

The project consists of a web server that displays health checks of 2 back-end servers, which is managed internally by a bastion host.

The project will utilise multiple technologies, such as:
  • Terraform
  • Ansible
  • Git & AWS CodePipeline
  • Vault
  • Docker

Throughout this project I will be making sure to follow best practices by making the sandbox secure through the usage of a bastion host and Hashicorp Vault, allow for automatic updates through Ansible and be highly available.

Please do check back for more details when it is completed...
